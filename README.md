# xylophone-app-ios
This app is a storyboard challenge on ios development

## Getting Started

### Clone into a new directory

```bash
git clone https://github.com/Divenesh/xylophone-app-ios.git
cd xylophone-app-ios
```

### Add to an existing local repository

If you already have a local directory and want to pull this repository into it:

```bash
cd /path/to/your/existing/directory
git init
git remote add origin https://github.com/Divenesh/xylophone-app-ios.git
git pull origin main
```

### Requirements

- Xcode (latest stable version recommended)
- iOS Simulator or a physical iOS device

### Running the App

1. Open the `.xcodeproj` or `.xcworkspace` file in Xcode.
2. Select a simulator or connected device.
